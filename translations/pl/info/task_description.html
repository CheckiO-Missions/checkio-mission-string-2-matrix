Otrzymują Państwo nieposortowany ciąg unikalnych (ignorując wielkość liter) małych i/lub wielkich liter. Proszę przedstawić go jako poprawną maskę dla kwadratowo ułożonych liter alfabetu od "a" do "y" (kolejność rząd-dur). Maska musi być macierzą 5x5 liczb całkowitych jako krotka krotek z następującą regułą.

  • znak nie występuje w łańcuchu - proszę użyć 0 w masce;
  • znak jest pisany małymi literami - proszę użyć 1;
  • znak jest pisany wielkimi literami - proszę użyć 2.

Oto misja, aby dokonać odwróconej konwersji - Matrix-2-String.
