Sie erhalten eine unsortierte Zeichenkette aus eindeutigen (Groß- und Kleinschreibung ignorierenden) Klein- und/oder Großbuchstaben. Stellen Sie sie als eine korrekte Maske für die quadratisch angeordneten Buchstaben des Alphabets von "a" bis "y" dar (zeilenweise Anordnung). Die Maske muss eine 5x5-Matrix von ganzen Zahlen als Tupel von Tupeln mit der folgenden Regel sein.

  • zeichen ist nicht in der Zeichenkette vorhanden - verwenden Sie 0 in der Maske;
  • das Zeichen wird klein geschrieben - verwenden Sie 1;
  • zeichen ist groß geschrieben - verwenden Sie 2.
